1\. Functional Requirements

Patient Management



FR-01: System shall allow registering new patients



FR-02: System shall let users search patients



FR-03: System shall store medical history



Appointment Management



FR-04: System shall allow booking appointments



FR-05: System shall show doctor availability



Medical Records



FR-06: Doctors shall create diagnosis records



FR-07: System shall store lab results



Billing



FR-08: System shall calculate charges automatically



FR-09: System shall support insurance payments



Pharmacy



FR-10: System shall track medicine stock



FR-11: System shall record prescriptions



Laboratory



FR-12: System shall allow test requests



FR-13: System shall upload test results



Admin



FR-14: System shall manage user accounts



FR-15: System shall generate reports



2\. Non-Functional Requirements



NFR-01: System must respond within 2 seconds



NFR-02: System must support 500+ users



NFR-03: System must ensure data privacy



NFR-04: System must provide 99% uptime



NFR-05: System must allow backup and recovery
