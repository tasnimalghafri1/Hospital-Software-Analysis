3\) user-stories.md

Patient User Stories



US-01: As a patient, I want to book an appointment online so I can avoid long waiting lines.



US-02: As a patient, I want to view my medical history so I can track my treatments.



Doctor User Stories



US-03: As a doctor, I want to access patient medical records so I can give proper treatment.



US-04: As a doctor, I want to update patient diagnosis so the system remains accurate.



Nurse User Stories



US-05: As a nurse, I want to view assigned patients so I can prepare treatment materials.



US-06: As a nurse, I want to record vital signs so doctors can review them.



Receptionist User Stories



US-07: As a receptionist, I want to register new patients quickly to reduce waiting times.



US-08: As a receptionist, I want to schedule appointments for walk-in patients.



Pharmacist User Stories



US-09: As a pharmacist, I want to receive prescriptions digitally to prepare the medicine.



US-10: As a pharmacist, I want to track medicine stock so I can reorder if needed.



Admin User Stories



US-11: As an admin, I want to manage user roles to control system access.



US-12: As an admin, I want to generate reports to monitor hospital performance.
